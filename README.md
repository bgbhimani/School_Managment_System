# School Management System API Documentation

- Working on the School Management System Backend using FastApi, SQLAlchemy, and PostgreSQL.


## How to Use this Project

### Installation Process
```cmd
pip install uv
uv add -r requirements.txt
```
- make Sure to have PostgreSQL installed and running on your machine.
- And Also Add the Database URL to the .env file

### Running the Application
```cmd
uvicorn app.main:app --reload
```
